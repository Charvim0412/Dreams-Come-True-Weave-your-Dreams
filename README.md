# 🌙 Lullaby — AI Sleep Companion

Lullaby is an AI-powered sleep companion that generates personalized bedtime experiences including stories, lullabies, affirmations, and calming soundscapes.

## 🚀 Features
- 🎧 AI-generated lullabies & bedtime stories
- 🧠 AI therapist for stress & anxiety relief
- 🌍 Multilingual support
- 🔊 Realistic voice generation
- 📚 Content library for saved experiences

## 🛠 Tech Stack
- Gemini API (content generation)
- ElevenLabs (text-to-speech)
- Firebase + Supabase (storage)
- Bolt (frontend/backend)
- Dappier SDK (AI assistant)

## 💡 Inspiration
Sleep issues affect a huge percentage of people due to stress and overthinking. Lullaby provides a calming, personalized experience to help users relax and sleep better.

## ⚙️ How to Run
1. Clone the repo  
2. Install dependencies  
3. Add API keys (.env)  
4. Run frontend and backend
